## Features implemented:

- Product search
- Filter by category
- Adding to favorites
- Responsive design
- Cart

## Getting Started

To get a local copy - follow these simple steps.

1. Clone the repo

```sh
  git clone https://github.com/Irina-Grebennikova/food-delivery-app.git
```

2. Install NPM packages

```sh
  npm install
```

3. Start project

```sh
  npm start
```
